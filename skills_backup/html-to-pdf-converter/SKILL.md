# HTML转PDF转换器

## 功能
专业的HTML到PDF转换助手，能够：
- 精确1:1转换HTML文件为PDF格式
- 保留所有CSS样式、布局和格式
- 支持图片、表格、链接等所有HTML元素
- 保持页面尺寸和分页正确
- 支持多页HTML文档转换

## 使用场景
- 需要将网页保存为PDF时
- 需要将HTML报告转换为PDF时
- 需要打印HTML内容但希望保留格式时
- 需要归档HTML文档为PDF格式时

## 转换能力

### 支持的功能
- 完整的CSS样式渲染
- 外部资源加载（图片、字体等）
- 多页文档处理
- 自定义页面尺寸
- 页眉页脚设置
- 方向设置（纵向/横向）

### 输出质量
- 1:1精确还原
- 矢量图形支持
- 高清图片保留
- 文本可选择和复制
- 链接保持可点击

## 工作流程
1. 接收HTML文件路径或HTML内容
2. 分析HTML结构和样式
3. 配置PDF转换参数
4. 执行转换过程
5. 验证转换结果
6. 输出PDF文件

## 输出格式
- 标准PDF文件格式
- 可选择保存路径
- 转换完成后的确认信息
- 可选的转换日志

## 转换方法
可以使用以下工具和方法进行转换：
- Puppeteer/Playwright（推荐，使用无头浏览器转换）
- wkhtmltopdf（传统工具）
- WeasyPrint（Python库）
- 其他专业HTML转PDF服务
