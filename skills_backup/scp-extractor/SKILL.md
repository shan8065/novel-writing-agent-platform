# SCP文档提取器

## 功能
从 e:\小说\SCP_PDF 目录下的 SCP 基金会 PDF 文档中智能提取纯文本内容，去除不必要的格式元素，节省 token 消耗：
- 提取 PDF 中的所有文本内容
- 去除 PDF 格式标签和样式
- 保持文本结构和段落
- 支持批量处理多个 PDF 文件

## 使用场景
- 需要给 AI 阅读 SCP 文档时
- 需要节省 token 消耗时
- 需要从 e:\小说\SCP_PDF 中提取多个 SCP 文档内容时
- 需要整理 SCP 文档资料库时

## 使用方法

### 基本使用
1. 扫描 e:\小说\SCP_PDF 目录
2. 选择或指定 PDF 文件路径
3. 提取 PDF 中的纯文本内容
4. 输出纯净的文本内容

### 高级选项
- 可以指定要提取的特定 PDF 文件
- 可以保存提取结果到文本文件
- 支持批量处理 e:\小说\SCP_PDF 中的所有 PDF 文件

## 输出格式
提取后的内容包含：
- 项目编号和等级
- 特殊收容措施
- 项目描述
- 附录和补充材料
- 所有正文文本

## 示例
输入：`SCP-6820 - SCP基金会.pdf`
输出：纯净的 SCP-6820 文档文本
