
# HTML转PDF转换工具使用说明

## 文件说明

- `convert_html_to_pdf.py` - 主要转换脚本
- `requirements.txt` - Python依赖包列表
- `使用说明.md` - 本文档

## 安装步骤

### 1. 安装Python依赖

在 `e:\小说` 目录下打开终端，运行：

```powershell
pip install -r requirements.txt
```

### 2. 安装Playwright浏览器

安装完依赖后，需要安装Playwright的Chromium浏览器：

```powershell
playwright install chromium
```

## 使用方法

### 方法一：直接运行脚本（推荐）

在 `e:\小说` 目录下打开终端，运行：

```powershell
python convert_html_to_pdf.py
```

脚本会自动：
1. 扫描 `e:\小说\SCP` 目录下的所有HTML文件
2. 逐个转换为PDF格式
3. PDF文件保存在同一目录下

### 方法二：通过智能体调用

创建好智能体后，可以直接让智能体执行转换任务。

## 智能体配置补充

在智能体的提示词中，可以添加以下内容让它能够使用这个脚本：

```
你可以使用 `e:\小说\convert_html_to_pdf.py` 脚本来批量转换HTML文件为PDF。

使用步骤：
1. 首先确保已安装依赖：`pip install playwright` 和 `playwright install chromium`
2. 运行脚本：`python e:\小说\convert_html_to_pdf.py`
3. 脚本会自动处理 `e:\小说\SCP` 目录下的所有HTML文件
```

## 特性

- ✅ 1:1还原HTML格式、样式和布局
- ✅ 保留所有图片资源
- ✅ 支持CSS样式和JavaScript渲染内容
- ✅ 批量自动转换
- ✅ 详细的转换进度显示
- ✅ 错误处理和统计

## 注意事项

1. 首次使用需要安装Playwright浏览器，可能需要几分钟
2. 转换过程中请确保网络连接正常（用于加载外部资源）
3. 大文件转换可能需要较长时间，请耐心等待
4. PDF文件会覆盖同名文件，请做好备份

## 故障排除

### 问题：找不到playwright模块
解决：运行 `pip install playwright`

### 问题：浏览器未安装
解决：运行 `playwright install chromium`

### 问题：图片无法加载
解决：确保HTML文件和对应的 `_files` 文件夹在同一目录下
